Snake Game - Python (Kivy)

This is a simple Snake Game built using Python and the Kivy framework. The objective of the game is to collect the red food items that appear on the screen while avoiding colliding with the borders or the snake's own body. The player can control the snake using the arrow keys and the score will increase with every food item collected.

Features:
- Basic Snake Game: Collect food and grow the snake.
- Game Over Popup: When the game ends, a pop-up will ask whether you want to play again or close the game.
- Customizable Colors: The background, snake, food, and score text are color-customized for a more pleasant visual experience.

Instructions:
- Use the arrow keys (Up, Down, Left, Right) to move the snake.
- When the snake collides with the border or itself, a game-over pop-up will appear.
- Click Play Again to reset the game or Close to exit.

Installation:
1. Clone this repository to your local machine:
   git clone https://github.com/pritish2403/SnakeGame.git

2. Install dependencies using pip:
   pip install kivy

3. Run the game:
   python snake_game.py
